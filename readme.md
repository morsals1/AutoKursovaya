# АвтоУчет (AutoKursovaya)

Мобильное приложение для учета расходов на автомобиль с аналитикой и напоминаниями.

## О проекте

"АвтоУчет" — это Android-приложение, помогающее автовладельцам отслеживать и анализировать расходы на содержание автомобиля. Приложение позволяет вести учет нескольких автомобилей, добавлять расходы по категориям, настраивать напоминания о техническом обслуживании, следить за износом шин и строить наглядные отчеты.

### Основные возможности
- Учет расходов по 10 категориям (топливо, обслуживание, шины и др.)
- Ведение нескольких автомобилей
- Аналитика с круговыми и столбчатыми диаграммами
- Умные напоминания (по дате, пробегу, периодические)
- Учет шин с прогнозированием износа
- Экспорт данных в CSV/JSON
- Резервное копирование и восстановление
- Работа без интернета (локальное хранение)

## Как запустить
1. Откройте проект в **Android Studio**.
2. Убедитесь, что у вас настроено устройство (эмулятор или физическое).
3. Нажмите кнопку **Запустить "приложение"** на панели инструментов.
4. Выберите целевое устройство и дождитесь завершения сборки.
